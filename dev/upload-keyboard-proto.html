Upload keyboard navigation proto

  • If the upload button is focused
    • Tab: Focuses a file menu item
  • If a file menu item is focused
    • Tab: Focuses the next tabbable element on the page
    • Shift-Tab: Focuses the upload button
    • Esc: Focuses the upload button
    • ArrowUp: Focuses the previous menu item
    • ArrowDown: Focuses the next menu item
    • ArrowRight: Focuses a file control button inside the item
  • If a file control button inside one file menu item is focused
    • Tab: Focuses the next tabbable element on the page
    • Shift-Tab: Focuses the file menu item
    • Esc: Focuses the file menu item
    • ArrowUp: Focuses the previous menu item
    • ArrowDown: Focuses the next menu item
    • ArrowRight: Focuses the next file control button
    • ArrowLeft: Focuses the previous file control button or the file menu item
